    Subsection Matrix Addition

    Definition 1. Matrix Addition.

    Let \(A = [a_{ij}]\) and \(B = [b_{ij}]\) be matrices in \(M_{m,n}\text{.}\) The sum of \(A\) and \(B\text{,}\) denoted \(A + B\text{,}\) is the matrix \(C = [c_{ij}]\) where \(c_{ij} = a_{ij} + b_{ij}\) for all \(1 \le i \le m\) and \(1 \le j \le n\text{.}\)

    Proposition 2. Additive Identity.

    Let \(A \in M_{m,n}\text{,}\) and let \(Z\) be the \(m \times n\) zero matrix, meaning each entry of \(Z\) is zero. Then \(A + Z = Z + A = A\text{.}\)
    ğŸ”—
    ğŸ”—

    Proof.

    For all \(i\) and \(j\text{,}\) \([A + Z]_{ij} = a_{ij} + 0 = a_{ij}\text{.}\) By definition of matrix equality, \(A + Z = A\text{.}\) The argument for \(Z + A\) is identical.

    Theorem 13. Consistency and Linear Combinations.

    A linear system \(A\mathbf{x} = B\) is consistent if and only if \(B\) can be expressed as a linear combination of the columns of \(A\text{.}\)
    ğŸ”—
    ğŸ”—

    Proof.

    If \(A\mathbf{x} = B\) has a solution \(\mathbf{x} = [c_j]\text{,}\) then \(B = \sum_j c_j \operatorname{col}_j(A)\text{.}\) Conversely, any such linear combination defines a solution.

    Subsection Matrix Inverses

    Definition 15. Matrix Inverse.

    Let \(A \in M_{n,n}\text{.}\) If there exists a matrix \(B\) such that \(AB = BA = I_n\text{,}\) then \(B\) is called the inverse of \(A\text{.}\) We will prove that the inverse is unique, and that means we can write it \(A^{-1}\) unambiguously (when it exists).
