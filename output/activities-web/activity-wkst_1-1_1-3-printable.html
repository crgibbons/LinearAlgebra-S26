    Theorem 7. Existence of the Additive Identity.

    The \(m \times n\) matrix of all zeros, \(Z\text{,}\) is the additive identity for the set of \(m \times n\) matrices.
    ğŸ”—
    That is for each \(A \in M_{m \times n}\text{,}\)
    ğŸ”—
    \begin{equation*} A + Z = Z + A = A. \end{equation*}
    ğŸ”—

    Proof.

    Let \(m, n\) be positive integers. Assume \(A \in M_{m\times n}\text{,}\) and assume \(Z \in M_{m \times n}\) is the matrix of all zeros.
    ğŸ”—
    Try find a way to use the definition of addition to get to the end of the proof (the next line). Since matrix equality happens at the entry level, you will probably have define \(A = [a_{i,j}]\) and \(Z = [z_{i,j}]\) for all \(1 \le i \le m\text{,}\) \(1 \le j \le n\text{,}\) and note that \(z_{i,j} = 0\) for each entry. Now use the definition of addition: what is the \(i,j\)-th entry of \(A+Z\text{?}\)
    ğŸ”—
    Therefore, \(A + Z = Z + A = A\text{.}\)
    ğŸ”—
    ğŸ”—
    Is \(Z^T = Z\text{?}\) How would you prove it? Unless \(Z\) is square, \(Z\) and \(Z^T\) have different sizes so they canâ€™t be equal. If \(Z\) is square (same number of rows and columns), then \(Z\) and \(Z^T\) have the same sizes and the same entries (all zeros), so they are equal.
